Rookie Route에 오신 것을 환영합니다

우리의 미션

커뮤니티 기반 학습과 멘토링을 통해 차세대 개발자들에게 힘을 실어주기

Rookie Route는 신입 개발자(루키)들이 커리어 경로를 탐색하고 전문적 성장을 가속화할 수 있도록 설계된 오픈소스 플랫폼입니다. 우리는 모든 개발자의 여정이 명확한 로드맵, 실용적인 멘토링, 그리고 실제 협업 기회를 통해 지원되어야 한다고 믿습니다.

왜 Rookie Route인가?

문제점

주니어 개발자들이 흔히 직면하는 문제들:

  • 🎯 다음에 무엇을 배워야 할지에 대한 압도적인 선택지
  • 🤔 커리어 발전 경로에 대한 불확실성
  • 💭 경험 많은 멘토에 대한 접근성 부족
  • 🔧 실제 프로젝트 경험 기회의 제한

우리의 솔루션

우리가 제공하는 것:

  • 📚 큐레이팅된 학습 로드맵 - 시니어 개발자들이 설계한 체계적인 경로
  • 🤝 멘토십 허브 - 진심으로 도움을 주는 경험 많은 개발자들과의 연결
  • 🚀 커뮤니티 프로젝트 - 실제 작업을 통한 오픈소스 협업 실습
  • 🤖 AI 기반 코드 분석 - 코드에 대한 즉각적인 피드백

시작하기

# 저장소 복제
git clone https://github.com/rookie-route/rookie-route-repository.git

# 프론트엔드 의존성 설치
cd app/frontend
npm install

# 백엔드 의존성 설치
cd ../backend
pip install -r requirements.txt

# 개발 시작!

포괄적인 문서에서 자세히 알아보세요.

기술 스택

  • 프론트엔드: React + Vite + Tailwind CSS
  • 백엔드: FastAPI + Python
  • 데이터베이스: MySQL + SQLAlchemy
  • AI 통합: OpenAI API
