## 1. Vision and Mission

* **Project Name**: Rookie Route
* **Mission**: To build an open-source, community-driven platform that guides new developers(Rookies) through their career paths and accelerates their professional growth.
* **Problem We Solve**: We aim to reduce the confusion and uncertainty faced by junior developers by providing curated learning roadmaps, practical mentorship, and real-world project contributions.

## 2. Core Features

* **Curated Roadmaps**: Learning paths designed by senior developers for various tech stacks.
* **Mentorship Hub**: A dedicated space for connecting Rookies with experienced mentors.
* **Community Projects**: Small, manageable tasks on our main repository that allow Rookies to practice open-source collaboration.

## 3. Technology Stack

* **Frontend**: **React** 
    * **Build Tool**: Vite
    * **State/Routing**: React Router DOM
    * **Styling**: Tailwind CSS
    * **Key Components**: Monaco Editor (@monaco-editor/react), Recharts 
* **Backend**: **Python / FastAPI**
    * **Database / ORM**: MySQL / SQLAlchemy
    * **AI Service**: OpenAI
